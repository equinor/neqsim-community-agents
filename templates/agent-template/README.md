# Agent Template

Use this template to create a new NeqSim Community Agent.

## Steps

1. Copy this directory to `agents/<agent-name>/`.
2. Replace placeholder names and descriptions.
3. Add required skills from NeqSim Community Skills.
4. Document inputs, outputs, assumptions, limitations, and validation checks.
5. Add realistic public examples.
6. Add example prompts.
7. Add tests or validation notes.
8. Run the repository tests.

Agents must assist engineers and must never claim to replace engineering judgement.
