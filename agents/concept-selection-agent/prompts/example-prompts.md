# Example Prompts

## Compare Two Concepts

```text
Use the Concept Selection Agent to compare a subsea tieback against a standalone
platform for a public synthetic gas opportunity on resource class, indicative
CAPEX/OPEX, discounted value, emissions, and step-out. Include assumptions and
human review requirements.
```

## Screen Decision Drivers

```text
Use the Concept Selection Agent to identify the key decision drivers and
sensitivities for two development concepts on a public example opportunity.
```

## Produce a Screening Ranking

```text
Use the Concept Selection Agent to produce a screening-level ranking of three
development concepts for a public synthetic opportunity and recommend validated
NeqSim field-development and economics studies. Include limitations and validation
steps.
```
