# Example Prompts

## Build And Report

```text
Use the TEG Dehydration Agent to build a TEG dehydration plant for the public synthetic default feed at 70 bara and 25 degC into an 85 bara, 35 degC absorber with 5500 kg/hr of 97 wt% lean TEG. Report the dry-gas water dew point, lean-TEG purity, and still-vent NMVOC and methane emissions. State assumptions and missing information.
```

## Emission Reduction Comparison

```text
Use the TEG Dehydration Agent to compare once-through versus recirculated stripping gas for a public example TEG plant. Report how the still-vent NMVOC, methane, and benzene emissions change and note the human review requirements.
```

## Sensitivity Screen

```text
Use the TEG Dehydration Agent to screen how reboiler temperature and stripping gas rate affect lean-TEG purity and dry-gas water dew point for a public example case. Summarize the trade-offs and recommended validated follow-up analysis.
```
