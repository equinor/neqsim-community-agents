# Example Prompts

These prompts use public, synthetic data only. They are intended for early-stage screening.

## Fire-Case Relief Load Screening

```text
Use public synthetic data to screen a fire-case relief load for a vessel with a wetted fire area of 50 m^2, latent heat of 300 kJ/kg, relief pressure of 20 bara, and no environment credit. Report the relief load indicator and warning level, and suggest validated API 520/521 and NeqSim workflows.
```

## Blowdown Time and Low-Temperature Screening

```text
Use public synthetic data to screen an emergency blowdown of a vessel from 100 bara to 7 bara with 2000 kg inventory, a representative blowdown rate of 12000 kg/h, a relieving temperature of 30 C, and an MDMT of -46 C. Report the blowdown time indicator and whether the cold-end temperature could fall below the MDMT.
```

## Combined Process Safety Screening

```text
Screen a separator segment for both a fire-case relief load and an emergency blowdown using public synthetic data. Summarize the relief load indicator, the blowdown time indicator, and the low-temperature flag. List the required follow-up studies and prepare a screening report outline with assumptions, limitations, and a human review checklist.
```
