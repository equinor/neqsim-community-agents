# Example Prompts

These prompts use public, synthetic data only. They are intended for early-stage screening.

## Route Profile and Pressure-Drop Screening

```text
Use public synthetic data to build the along-route elevation profile from ordered waypoints, then estimate the single-phase pressure drop and compare it against a recommended gradient of 0.25 bar/km. Report the total route length, the maximum slope, the pressure-drop indicator, and the warning level.
```

## Velocity and Hydrate-Margin Screening

```text
Use public synthetic data to screen a line for velocity against erosional and recommended limits, and to screen the hydrate temperature margin given an operating temperature of 8 degC and a hydrate equilibrium temperature of 14 degC against a 3 degC guideline. Report each indicator and its warning level, and suggest validated NeqSim workflows.
```

## Combined Pipe Route Screening

```text
Screen a candidate pipe route using public synthetic data. Build the elevation profile, estimate the pressure drop against the recommended gradient, screen the line velocity, and screen the hydrate margin. List the required follow-up studies and prepare a screening report outline with assumptions, limitations, and a human review checklist.
```
