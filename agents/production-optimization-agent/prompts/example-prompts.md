# Example Prompts

## Frame an Optimization Problem

```text
Use the Production Optimization Agent to frame a setpoint-search problem for a
public synthetic separation-and-compression train: list decision variables,
bounds, objective, and constraints. Include assumptions and human review requirements.
```

## Screen a Setpoint Change

```text
Use the Production Optimization Agent to screen whether raising first-stage
separation pressure by 5 bar increases export throughput on a public example
train without pushing the export compressor outside its operating window.
```

## Rank Operating Directions

```text
Use the Production Optimization Agent to rank candidate operating directions for
minimizing export compression power on a public synthetic train, and recommend a
validated NeqSim optimization run. Include limitations and validation steps.
```
