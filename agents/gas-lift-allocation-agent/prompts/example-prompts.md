# Example Prompts

## Frame the Allocation Problem

```text
Use the Gas Lift Allocation Agent to frame a lift-gas allocation problem for a
public synthetic set of three gas-lifted wells with a fixed lift-gas budget. List
wells, objective, and constraints. Include assumptions and human review requirements.
```

## Screen a Lift-Gas Split

```text
Use the Gas Lift Allocation Agent to screen whether moving lift gas from a
low-productivity well to a higher-productivity well increases total production on
a public example well set.
```

## Rank Allocation Directions

```text
Use the Gas Lift Allocation Agent to rank candidate lift-gas splits for a public
synthetic well set and recommend a validated NeqSim production-optimization run.
Include limitations and validation steps.
```
