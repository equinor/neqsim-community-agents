# Example Prompts

## Identify the Bottleneck

```text
Use the Debottlenecking Agent to screen which unit limits a 15% throughput
increase on a public synthetic separation-and-compression train. Rank units by
approximate utilization. Include assumptions and human review requirements.
```

## Screen a Line

```text
Use the Debottlenecking Agent to screen a public synthetic export line for
velocity and pressure-drop headroom at a higher flow rate, and state whether the
line or upstream equipment is the limiting constraint.
```

## Suggest Debottlenecking Directions

```text
Use the Debottlenecking Agent to suggest candidate debottlenecking directions for
the limiting unit on a public example train, and recommend a validated NeqSim
capacity study. Include limitations and validation steps.
```
