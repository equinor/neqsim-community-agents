# Example: Process Screening Checklist

This example uses a public synthetic inlet separation concept for demonstration only.

## Input

- Process: two-phase inlet separation before gas export and liquid handling
- Inlet pressure: 60 bara
- Inlet temperature: 35 degC
- Feed: methane-rich hydrocarbon stream with possible condensate
- Screening objective: identify initial simulation and study needs

## Expected Agent Actions

1. Review the process description and identify missing stream data.
2. Identify separator screening questions such as gas rate, liquid loading, residence time, and operating range.
3. Recommend public NeqSim workflow examples for fluid setup, TP flash, and simple separator modelling.
4. Generate an engineering checklist and follow-up study list.

## Example Output Summary

- Major missing data includes composition, flow rate range, liquid fraction, water content, turndown, slugging assumptions, and product specifications.
- Recommended simulations include fluid setup, phase split checks, inlet condition variation, and separator sensitivity checks.
- Follow-up studies should include process simulation, flow assurance, control philosophy, relief and safety review, and operability review.

## Human Review

A qualified process engineer must review assumptions, simulations, equipment limits, and safety implications before engineering use.
