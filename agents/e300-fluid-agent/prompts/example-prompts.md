# Example Prompts

## Read E300 into NeqSim

```text
Use the E300 Fluid Agent to read this E300 file into a NeqSim fluid. List the
components and EOS, set the classic mixing rule, and explain what needs human
review before simulation.
```

## Add Water and Write E300

```text
Use the E300 Fluid Agent to add water to this E300 file with the PVTsim water
parameters, write a *_water.e300 file, and confirm it reloads in NeqSim. List
the water parameters used and the assumptions and limitations.
```

## Write E300 from a NeqSim Fluid

```text
Use the E300 Fluid Agent to write this NeqSim fluid to an E300 file at a stated
reservoir temperature. Show the E300 keyword text and include a validation
checklist.
```
